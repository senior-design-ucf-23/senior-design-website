Project Description

This is a self-funded project, through which we are interested in converting a traditional bicycle into a technologically updated electric bicycle. The motivation behind this project is to acquire the skills and knowledge to perform the conversion; the ability to implement the knowledge we're learning here will be valuable to our future endeavors.
